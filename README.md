# Mini Calculator (Mobile Style)
## Description
This is a mobile-style calculator frontend built using HTML, CSS, and JavaScript.  
It performs basic arithmetic operations with a user-friendly interface similar to a smartphone calculator.
## Technologies Used
- HTML
- CSS
- JavaScript
## Features
- Mobile-like calculator UI
- Addition, Subtraction, Multiplication, Division
- Clear and delete functionality
- Instant result display
